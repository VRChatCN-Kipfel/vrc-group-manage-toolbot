## 💡 使用建议

1. **新用户**：先阅读主 README 了解项目概况
2. **需要 HTML 渲染功能**：查看 `HTML_RENDER.md` 了解接口定义与主题配置
3. **需要定时任务功能**：查看 `SCHEDULER_USAGE.md` 了解任务类型与管理方法
4. **实际开发**：参考相应文档中的"快速开始"与"高级技巧"章节
5. **遇到问题**：查阅指南中的"故障排除"部分
